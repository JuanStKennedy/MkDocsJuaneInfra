# Diagrama General de la Topología 

A continuación se muestra la topología simulada en **GNS3**, la cuál tiene conectividad mediante **Tailscale** a una
máquina virtual de AWS y OCI (Oracle Cloud Infraestructure).  
La misma incluye un router Cisco y VyOS, switches Cisco, un firewall PfSense y servidores Debian 12.
También se muestra con un icono el servicio de docker desplegado en cada server, el servidor de bastión y el Samba DC.
